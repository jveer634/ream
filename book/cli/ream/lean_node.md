# ream lean_node

Start the lean node

```bash
$ ream lean_node --help
```
```txt
Usage: ream lean_node [OPTIONS] --network <NETWORK>

Options:
  -v, --verbosity <VERBOSITY>  Verbosity level [default: 3]
      --network <NETWORK>      Provide a path to a YAML config file
  -h, --help                   Print help
```
